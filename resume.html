Electron · CoreML · Google Gemini · React Flow · macOS + Windows · 2025
Professional AI visualisation tool for architects. Node-based Variation Canvas (React Flow) enables structured, branching image generation — users wire reference inputs (sketches, site photos, material samples) into a generation graph, producing parallel architectural variations. C# · HLSL · Unity · Open Source · 2020–2021
Real-time global illumination plugin for Unity. Monte-Carlo cone tracing over sparse voxel octrees for diffuse, specular, and shadow lighting. Baked and real-time voxelisation, runtime streaming, multi-threading, volumetric lighting, VR single-pass stereo. Co-developed remotely with graphics researcher Abigail Hocking.
